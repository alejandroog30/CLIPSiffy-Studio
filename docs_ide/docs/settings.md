# Settings & Customization

The `Settings` menu allows you to optimize the IDE's performance:

- **Generate Inference Graphs**: Checked by default. Uncheck this if you are running massive simulations with hundreds of rules and want to speed up the `(run)` process by skipping the Matplotlib rendering.

- **Enable Fuzzy Logic Mode**: Checked by default. Uncheck to hide the Fuzzy Builder, Fuzzy Facts, and Block Diagram panels. This turns the application into a lightweight, standard CLIPS IDE and stops injecting the background fuzzy library.

- **Generate Block Diagrams**: Checked by default. Uncheck to disable the topological calculation and rendering of the block canvas, saving memory and processing power during complex system loads.
