# Keyboard Shortcuts Cheat Sheet

| Action | Shortcut |
| :--- | :--- |
| **File Operations** | |
| New File | `Ctrl+N` |
| Open File | `Ctrl+O` |
| Open Folder | `Ctrl+K, Ctrl+O` |
| Save | `Ctrl+S` |
| Save As... | `Ctrl+Shift+S` |
| Export Fuzzy Config | `Ctrl+E` |
| Exit | `Ctrl+F4` |
| **Editing** | |
| Undo / Redo | `Ctrl+Z` / `Ctrl+Y` |
| Cut / Copy / Paste | `Ctrl+X` / `Ctrl+C` / `Ctrl+V` |
| Find / Find Next | `Ctrl+F` / `F3` |
| Toggle Comment | `Ctrl+/` |
| **View** | |
| Zoom In / Zoom Out | `Ctrl++` / `Ctrl+-` |
| Clear Console | `Ctrl+L` |
| **Execution** | |
| Load Environment | `F4` |
| Reset | `F5` |
| Run | `F6` |
| Step | `F7` |
| Clear Environment | `Ctrl+Shift+L` |
| **Help** | |
| Fuzzy Library Docs | `F1` |
